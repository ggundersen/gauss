Problem statement

If we list all the natural numbers below 10 that are multiples of 3 or 5, we get 3, 5, 6 and 9. The sum of these multiples is 23.

Find the sum of all the multiples of 3 or 5 below 1000.

Analysis

The solution has O(n) complexity. The limit, 'below 1000', is what impacts runtime. If it increases twofold or tenfold, so does runtime.

Input

Input the limit. You should note that runtime increases linearly with input.
