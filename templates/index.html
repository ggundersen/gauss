Gauss

Intro

Gauss is a web application for running, analyzing, and documenting Project Euler algorithms. All calculations are performed at runtime.

The problems

The primary goal of Gauss is to showcase the significant variables that effect the efficiency of various algorithms. In other words, Gauss should help us internalize the concepts of time complexity.

Get started here.

The API

Gauss has a simple API that allows users to run and collect the data themselves. For example, the below API call http://project-gauss.appspot.com/api/q=pe1&input=1 will return the following data {
'problem': 1,
'answer': 233168,
'input': 1,
'runtime': 0.01
 } When the user input is blank or matches the question's variables exactly, the answer is not returned. This is in accordance with Project Euler's honor code.

Credits

Gauss is being developed on the Google App Engine platform. The application uses the webapp2 framework, Python 2.7, and jinja2.
