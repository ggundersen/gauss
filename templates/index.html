Gauss

Intro

Gauss is a web application for testing, analyzing, and documenting Project Euler solutions and associated libraries. All calculations are performed at runtime.

Run the test suite.

The gmath library

While Project Euler problems are standalone, many of them rely on common mathematical functions. These recurring functions are abstracted away by the gmath library, resulting in cross-problem performance gains. A trivial example: def factorial(n):
if n == 1:
return 1
 else:
return n * factorial(n-1) was refactored to be def get_factorial(n):
facts = [1, 1]
for i in range(2, n+1):
facts.append(facts[i-1] * i)
 return facts[n] The latter is orders of magnitude faster for large values of n and optimized multiple solutions.
