Gauss

Intro

Gauss is a web application for testing, analyzing, and documenting Project Euler solutions and the utility library, gmath.

For each problem, Gauss runs and times the algorithm, validates the answer, and renders the JSON response. Correct answers are masked, and Project Euler's "one-minute rule" is conveniently reinforced by Google App Engine, which kills any request that takes longer than 60 seconds.

The API

Gauss offers an API in which problem data can be queried in groups. This allows fast problems to be executed together, avoiding blocking from slower problems and minimizing delays due to Internet speeds.

For example: /api/problems=1,2,5 will return the following JSON object (runtimes vary): {

The gmath library

While Project Euler problems are standalone, many of them rely on common mathematical functions. These recurring functions are abstracted away by the gmath library, resulting in cross-problem performance gains. For example: def factorial(n):
if n == 1:
return 1
 else:
return n * factorial(n-1) was refactored to be def get_factorial(n):
facts = [1, 1]
for i in range(2, n+1):
facts.append(facts[i-1] * i)
 return facts[n] The latter is orders of magnitude faster for large values of n and optimized multiple solutions.
