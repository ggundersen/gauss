Gauss

Intro

Gauss is a web application for standardizing, testing, and analyzing Project Euler solutions. All calculations are performed at runtime.

The gmath library

While Project Euler problems are standalone, many of them rely on common mathematical functions. These recurring functions are abstracted away by the gmath library, resulting in cross-problem performance gains. For example: def factorial(n):
if n == 1:
return 1
 else:
return n * factorial(n-1) was eventually rewritten to be: def get_factorial(n):
facts = [1, 1]
for i in range(2, n+1):
facts.append(facts[i-1] * i)
 return facts[n] The latter is orders of magnitude faster for larger values of n and improved multiple problems.

A mathematical foundation

Many problems have trivial, brute-force "solutions" that fail due to hitting a computational wall. The primary thrill in solving such problems is discovering that a seemingly unrelated mathematical idea removes the wall entirely. These discoveries are documented.
